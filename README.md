# Вопросы, с которыми я столкнулся
## TeamMember 
Не сказано может ли быть участник привязан к нескольким командам
Допущение: Судя по запросам человек может быть только в одной команде



## UpdateUser in a team
Самое странное, что судя по описанию будто бы мы можем обновлять пользователей в команде без админских прав, и токена никакого не нужно

Есть два пути: 1. запретить обновлять активность пользователя в команде 2. разрешить обновлять, но при наличии токена


## Error for team
Ошибка 400 возвращается, только тогда подаем пустую существующую команду без участников
Иначе создаем команду если нет, обновляем пользователей (добавляем новых, обновляемых старых)
